Description
Ecommerce Plugin allows the handling of products with prices and online payments via PayPal or other payment method, Ecommerce plugin has the following characteristics:

  • Order Management
  • Manage Tax Rates
  • Handling shipping methods
  • Handling Payment Methods
  • Management Coupons
  • Currency settings type and type Weight
